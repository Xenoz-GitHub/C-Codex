
# C++ Codex

> A comprehensive single-page HTML reference for mastering C++ syntax.  
> Dark cyber theme · Fully responsive · Zero JavaScript.

## Overview

C++ Codex is a distilled syntax reference covering every major C++ construct — from fundamental types to C++20 concepts. Each topic includes a concise explanation and a runnable-minded example, designed for quick lookup and memorization.

## Features

- **49 syntax sections** organized into 8 logical categories
- **Pure HTML + CSS** — no JavaScript, no dependencies, no build step
- **Dark cyber theme** — neon cyan and magenta accents on a deep navy background
- **Responsive** — fluid layout adapts from 290px to ultrawide displays
- **Syntax-highlighted code blocks** — color-coded keywords, types, strings, comments, and numbers
- **Zero-config deployment** — open `index.html` directly or host on any static server

## Sections

| # | Category | Topics |
|---|----------|--------|
| 01 | **Basics** | Comments, Variables & Types, I/O, Operators, Enum & Struct, Type Casting, Namespaces, Type Aliases, Raw Strings |
| 02 | **Control Flow** | If/Else, Switch, For & While, Range-based For, Recursion |
| 03 | **Data & Memory** | Arrays & Strings, Pointers & References, Dynamic Memory |
| 04 | **Functions** | Functions, Lambdas, Function Pointers, Inline & constexpr, =default & =delete, decltype & typeid |
| 05 | **OOP** | Classes, Inheritance, Polymorphism, Copy Control, Operator Overloading, Static, Friend, explicit, Conversion Operators |
| 06 | **Modern C++** | Templates, Exceptions, STL Containers, Move Semantics, Iterators & Algorithms, optional & variant, Structured Bindings, pair & tuple, chrono, string_view, if constexpr, Fold Expressions |
| 07 | **System & Build** | File I/O, Preprocessor, Const Correctness, Union & Bit Fields |
| 08 | **Advanced** | Lvalues & Rvalues, Concepts (C++20), Placement new |

## Usage

1. Download [`index.html`](./index.html)
2. Open in any browser — no build tools required
3. Navigate using the sticky top bar, or scroll through categories
